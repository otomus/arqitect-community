# docx_read

Read text content from a DOCX (Microsoft Word) file using python-docx.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| file_path | string | yes | Path to the DOCX file to read |

## Usage

```python
text = run("/path/to/document.docx")
```

## Dependencies

- `python-docx` — install with `pip install python-docx`
