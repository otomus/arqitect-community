# pptx_read

Read text content from a PowerPoint PPTX file using python-pptx.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| file_path | string | yes | Path to the PowerPoint file to read |

## Usage

```python
text = run("/path/to/presentation.pptx")
```

## Dependencies

- `python-pptx` — install with `pip install python-pptx`
