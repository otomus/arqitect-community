# pdf_read

Read text content from a PDF file using pdfplumber.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| file_path | string | yes | Path to the PDF file to read |

## Usage

```python
text = run("/path/to/document.pdf")
```

## Dependencies

- `pdfplumber` — install with `pip install pdfplumber`
