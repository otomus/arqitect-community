# markdown_to_html

Convert Markdown text to HTML using the markdown library.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| markdown_text | string | yes | Markdown text to convert to HTML |

## Usage

```python
html = run("# Hello World\n\nSome **bold** text.")
```

## Dependencies

- `markdown` — install with `pip install markdown`
