# youtube_info

Get detailed information about a YouTube video using yt-dlp (no API key required).

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| video_id | string | Yes | YouTube video ID (e.g. dQw4w9WgXcQ) |

## Dependencies

- `yt-dlp` - Install with `pip install yt-dlp`

## Usage

```python
result = run("dQw4w9WgXcQ")
```
