# perplexity

Perplexity AI — search-augmented generation with real-time knowledge.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y perplexity-mcp-server
```

## Authentication

Requires a Perplexity API key. Set the `PERPLEXITY_API_KEY` environment variable.

## Tools

- search
- ask

## Capabilities

`search` `qa` `research` `knowledge`
