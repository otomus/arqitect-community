# obsidian

Obsidian knowledge graph notes.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y obsidian-mcp-server
```

## Authentication

No authentication required.

## Tools

- `search_notes` — Search through notes
- `read_note` — Read an existing note
- `create_note` — Create a new note
- `update_note` — Update an existing note
- `list_notes` — List all notes

## Capabilities

`notes` `knowledge_graph` `markdown` `writing` `pkm`
