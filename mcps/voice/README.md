# voice

Voice wake word and talk mode.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y voice-mcp-server
```

## Authentication

No authentication required.

## Tools

- start_listening
- stop_listening
- text_to_speech
- speech_to_text

## Capabilities

`voice` `speech` `tts` `stt` `wake_word`
