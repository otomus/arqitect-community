# gmail

Gmail email with Pub/Sub triggers.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y gmail-mcp-server
```

## Authentication

Requires OAuth2 authentication with google.

## Tools

- send_email
- search_emails
- read_email
- list_labels
- watch_inbox

## Capabilities

`email` `gmail` `send` `inbox` `notifications`
