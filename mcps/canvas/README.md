# canvas

Visual workspace and app-to-UI generation.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y canvas-mcp-server
```

## Authentication

No authentication required.

## Tools

- create_canvas
- draw
- render_component
- export_image

## Capabilities

`visual` `drawing` `ui_generation` `design` `workspace`
