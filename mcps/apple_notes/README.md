# apple_notes

Apple Notes on macOS.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y apple-notes-mcp-server
```

## Authentication

No authentication required.

## Tools

- `search_notes` — Search through Apple Notes
- `create_note` — Create a new note
- `read_note` — Read an existing note
- `update_note` — Update an existing note

## Capabilities

`notes` `writing` `organization` `apple` `macos`
