# gif_search

GIF search via Giphy and Tenor.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y gif-search-mcp-server
```

## Authentication

Requires an API key via the `GIPHY_API_KEY` environment variable.

## Tools

- search_gifs
- trending_gifs
- random_gif

## Capabilities

`gif` `animation` `meme` `media` `fun`
