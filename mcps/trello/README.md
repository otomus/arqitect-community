# trello

Trello kanban boards.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y trello-mcp-server
```

## Authentication

Requires an API key. Set the `TRELLO_API_KEY` environment variable.

## Tools

- `list_boards` — List all boards
- `list_cards` — List cards on a board or list
- `create_card` — Create a new card
- `move_card` — Move a card to a different list
- `add_comment` — Add a comment to a card

## Capabilities

`kanban` `boards` `cards` `project_management` `tasks`
