# shazam

Shazam song recognition.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y shazam-mcp-server
```

## Authentication

Requires an API key. Set the `SHAZAM_API_KEY` environment variable.

## Tools

- `identify_song` — Identify a song from audio
- `search_songs` — Search for songs

## Capabilities

`music` `recognition` `identify` `song` `audio`
