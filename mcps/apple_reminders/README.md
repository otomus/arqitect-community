# apple_reminders

Apple Reminders on macOS.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y apple-reminders-mcp-server
```

## Authentication

No authentication required.

## Tools

- `list_reminders` — List all reminders
- `create_reminder` — Create a new reminder
- `complete_reminder` — Mark a reminder as complete
- `search_reminders` — Search through reminders

## Capabilities

`tasks` `reminders` `todo` `apple` `macos`
