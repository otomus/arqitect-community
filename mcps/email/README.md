# email

General email via SMTP and IMAP.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y email-mcp-server
```

## Authentication

Requires an API key via the `EMAIL_CREDENTIALS` environment variable.

## Tools

- send_email
- read_emails
- search_emails
- list_folders

## Capabilities

`email` `smtp` `imap` `send` `inbox`
