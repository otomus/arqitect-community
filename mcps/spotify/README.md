# spotify

Spotify music playback.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y spotify-mcp-server
```

## Authentication

Requires OAuth2 authentication with Spotify.

## Tools

- `play` — Start or resume playback
- `pause` — Pause playback
- `next` — Skip to the next track
- `previous` — Go back to the previous track
- `search` — Search for tracks, albums, or artists
- `queue` — Add a track to the queue
- `now_playing` — Get the currently playing track

## Capabilities

`music` `playback` `playlist` `streaming` `audio`
