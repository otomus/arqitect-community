# sonos

Sonos multi-room audio.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y sonos-mcp-server
```

## Authentication

No authentication required.

## Tools

- `play` — Start or resume playback
- `pause` — Pause playback
- `volume` — Set volume level
- `list_rooms` — List available rooms
- `group_rooms` — Group rooms together
- `now_playing` — Get the currently playing track

## Capabilities

`music` `audio` `multiroom` `speakers` `smart_home`
