# cron

Scheduled task execution.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y cron-mcp-server
```

## Authentication

No authentication required.

## Tools

- create_schedule
- list_schedules
- delete_schedule
- trigger_now

## Capabilities

`scheduling` `cron` `timer` `automation` `recurring`
