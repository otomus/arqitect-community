# onepassword

1Password secure credentials.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y 1password-mcp-server
```

## Authentication

Requires an API key via the `OP_SERVICE_ACCOUNT_TOKEN` environment variable.

## Tools

- get_item
- list_items
- list_vaults
- search_items

## Capabilities

`passwords` `secrets` `credentials` `security` `vault`
