# webhooks

External trigger endpoints.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y webhooks-mcp-server
```

## Authentication

No authentication required.

## Tools

- create_webhook
- list_webhooks
- delete_webhook
- get_webhook_logs

## Capabilities

`webhooks` `triggers` `http` `automation` `events`
