# peekaboo

Screen capture and control.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y peekaboo-mcp-server
```

## Authentication

No authentication required.

## Tools

- capture_screen
- capture_window
- list_windows
- ocr_screen

## Capabilities

`screenshot` `screen_capture` `ocr` `vision` `desktop`
