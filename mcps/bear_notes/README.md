# bear_notes

Bear markdown notes.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y bear-notes-mcp-server
```

## Authentication

No authentication required.

## Tools

- `search_notes` — Search through notes
- `create_note` — Create a new note
- `read_note` — Read an existing note
- `add_tag` — Add a tag to a note

## Capabilities

`notes` `markdown` `writing` `tags` `macos`
