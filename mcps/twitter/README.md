# twitter

Twitter/X integration.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y twitter-mcp-server
```

## Authentication

Requires an API key via the `TWITTER_BEARER_TOKEN` environment variable.

## Tools

- post_tweet
- search_tweets
- get_timeline
- reply_tweet
- get_user

## Capabilities

`twitter` `social` `tweets` `posting` `search`
