# eight_sleep

8Sleep smart mattress.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y eight-sleep-mcp-server
```

## Authentication

Requires an API key. Set the `EIGHT_SLEEP_TOKEN` environment variable.

## Tools

- `get_sleep_data` — Get sleep tracking data
- `set_temperature` — Set bed temperature
- `get_status` — Get current mattress status
- `set_alarm` — Set a smart alarm

## Capabilities

`sleep` `temperature` `health` `smart_home` `wellness`
