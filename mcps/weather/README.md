# weather

Weather forecasts and conditions.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y weather-mcp-server
```

## Authentication

Requires an API key via the `WEATHER_API_KEY` environment variable.

## Tools

- get_forecast
- get_current
- get_alerts
- get_hourly

## Capabilities

`weather` `forecast` `temperature` `conditions` `climate`
