# image_gen

AI image generation.

## Installation

This MCP server is installed automatically by arqitect-core. To use it manually:

```bash
npx -y image-gen-mcp-server
```

## Authentication

Requires an API key via the `IMAGE_GEN_API_KEY` environment variable.

## Tools

- generate_image
- edit_image
- describe_image

## Capabilities

`image_generation` `art` `creative` `dall_e` `stable_diffusion`
